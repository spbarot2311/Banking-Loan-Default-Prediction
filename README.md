# Banking-Loan-Default-Prediction
Build a predictive model to identify high-risk vehicle loan applicants, enabling improved credit risk assessment and  reduced financial losses.
This project focused on developing a predictive model to identify high-risk vehicle loan applicants, helping improve credit risk assessment and reduce potential financial losses.
I worked with real-world-style loan data using Python libraries such as Pandas and NumPy for data manipulation, and Seaborn for visualization. I started by cleaning and preprocessing the dataset, which included handling missing values, removing duplicates, and standardizing variables to ensure data quality.
The insights from this project can support financial institutions in making better lending decisions by highlighting high-risk applicants and improving overall risk management strategies.
